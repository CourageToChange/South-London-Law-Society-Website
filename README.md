# South London Law Society — Case Management Website

A full-stack web application for the South London Law Society: members of the public
submit legal cases online, solicitors browse and take on cases, and administrators
oversee users and casework.

## Features

- **Streamlined case submission** — structured intake form for new legal cases
- **Real-time case tracking** — applicants can follow the status of their case
- **Solicitor workflow** — case browsing, selection and management for solicitors
- **Admin controls** — user and case oversight for administrators
- **Security-minded design** — sensitive case data protected with AES encryption

## Tech stack

| Layer | Technology |
|---|---|
| Front end | React.js |
| Back end | Node.js + Express.js |
| Database | MySQL |
| Data protection | AES encryption |
